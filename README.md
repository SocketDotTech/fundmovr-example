# Fund-movr-example
> Example for using fund-movr-sdk for sending tokens from one chain to other

----

> This example sends 1 USDC coin from polygon mainnet to xDai mainnet

## Installation
> ``` npm install ```

## start
> 1. Set .env from .env.sample
> 2. ``` node index.js ```
